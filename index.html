TA_B401_Aryaduta

Tugas akhir ini akan mengeksplor penerapan Model Bahasa Besar (Large Language model) yang telah di fine tuned menggunakan LoRA (Low Rank Adaptation) dengan Grounding Konteks Multimodalitas untuk menghasilkan rencana aksi robotik (Robotic Plan).

Robotic Plan Generation with LoRa-Tuned LLM and Multimodal Grounding Context

Platform yang digunakan dalam proyek ini meliputi dobot magician sebagai robot lengan, gemma sebagai model bahasa, unsloth sebagai library untuk melakukan tuning, huggingface sebagai repository dataset dan model bahasa.
